# 📚 智能單字卡應用程式

一個基於 Streamlit 的智能單字卡學習工具,支援滑動操作和智能頻率調整。

## 功能特色

- ✅ **左右滑動評分**: 向左滑標記不熟,向右滑標記很熟
- 🎯 **智能出現頻率**: 根據熟悉度自動調整單字出現機率
- ➕ **快速新增生字**: 底部輸入欄位可隨時新增新單字
- 📊 **學習統計**: 即時顯示學習進度和單字熟悉度
- 💾 **自動儲存**: 所有學習進度自動保存

## 部署到 Streamlit Cloud

### 步驟 1: 上傳到 GitHub

1. 在 GitHub 建立新的 repository
2. 將以下檔案上傳到 repository:
   - `app.py`
   - `requirements.txt`
   - `README.md`

### 步驟 2: 部署到 Streamlit Cloud

1. 前往 [streamlit.io/cloud](https://streamlit.io/cloud)
2. 使用 GitHub 帳號登入
3. 點擊 "New app"
4. 選擇你的 repository
5. 主檔案路徑設為 `app.py`
6. 點擊 "Deploy"

## 本地運行

```bash
# 安裝依賴
pip install -r requirements.txt

# 運行應用程式
streamlit run app.py
```

## 使用說明

1. **查看單字**: 應用程式會自動顯示一個單字
2. **顯示翻譯**: 點擊「顯示/隱藏翻譯」按鈕
3. **評分**:
   - 點擊「⬅️ 不熟」: 降低熟悉度,增加未來出現頻率
   - 點擊「➡️ 很熟」: 提高熟悉度,減少未來出現頻率
4. **新增生字**: 在底部輸入欄位填寫英文單字和中文翻譯,點擊「新增單字」

## 熟悉度系統

- **範圍**: -10 (很不熟) 到 +10 (很熟)
- **權重計算**: 熟悉度越低,出現機率越高
- **每次評分**: 不熟 -2 分 / 很熟 +2 分

## 資料儲存

所有單字和學習進度儲存在 `vocabulary_data.json` 檔案中,會自動保存。

## 技術架構

- **前端框架**: Streamlit
- **資料格式**: JSON
- **部署平台**: Streamlit Cloud
- **版本控制**: GitHub

## 自訂功能

你可以根據需求修改以下參數:

- 熟悉度調整幅度 (目前為 ±2)
- 熟悉度範圍 (目前為 -10 到 +10)
- 權重計算公式 (在 `calculate_weight` 函數中)
- 初始單字列表 (在 `load_data` 函數中)
